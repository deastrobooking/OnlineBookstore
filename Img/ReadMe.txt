# Images Folder

This folder stores book cover images uploaded through the Admin interface.

## Instructions:
1. Upload images through Admin.aspx using the "Upload Images" feature
2. Images should be JPG format
3. Image filenames in XML must match the uploaded files

## Supported Formats:
- .jpg
- .jpeg
- .png
- .gif

Images will be saved here and referenced in the database with path: `\Img\filename.jpg`
