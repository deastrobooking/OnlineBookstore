# App_Data Folder

This folder stores application data including:

1. **ownerID.txt** - Stores the logged-in user's owner ID (created automatically on login)
2. **BobsBookNook.mdf** - SQL Server LocalDB database file (created automatically on first run)
3. **sample-books.xml** - Sample XML data file for testing the upload feature

## Database:
The database file will be created automatically when you first run the application.
Table structure: {ownerID}_BOOKS

## XML Format:
See sample-books.xml for the correct XML structure to upload book data.
